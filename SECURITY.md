# Security Policy

## Supported Versions

| Version | Supported |
|----------|-----------|
| 1.x.x | ✅ |
| < 1.0 | ❌ |

---

## Reporting a Vulnerability

If you discover a security issue, please open a GitHub Issue with:

- Description
- Steps to reproduce
- Expected behavior
- Screenshots (if applicable)

We appreciate responsible disclosure and will review reports promptly.
