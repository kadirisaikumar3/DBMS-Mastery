# 🔑 Key Concepts

## Database

Organized collection of data.

---

## DBMS

Software used to manage databases.

---

## Table

Collection of rows and columns.

---

## Row (Record)

Single entry in a table.

Example:

| ID | Name |
|----|------|
| 1 | Sai |

---

## Column (Field)

Represents an attribute.

Example:

Name

Age

Salary

---

## Primary Key

Uniquely identifies each record.

Example:

StudentID

---

## Query

Request for information from database.

Example:

SELECT * FROM Students;

---

## Metadata

Data about data.
