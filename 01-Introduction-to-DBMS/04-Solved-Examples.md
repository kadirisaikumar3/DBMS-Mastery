## Example 1

Question:

What is DBMS?

Answer:

DBMS is software used to create and manage databases efficiently.

Examples:

MySQL, Oracle, PostgreSQL.

## Example 2

Question:

Difference between Data and Information?

Answer:

Data = Raw facts.

Information = Processed data.

Example:

Data: 23

Information: Age = 23
