# ❌ Common Mistakes

## Mistake 1

Confusing Database and DBMS

Database = Data

DBMS = Software

---

## Mistake 2

Confusing Row and Column

Row = Record

Column = Field

---

## Mistake 3

Thinking DBMS stores only tables

DBMS stores:

- Tables
- Views
- Indexes
- Procedures

---

## Mistake 4

Giving only definitions in interviews

Always provide:

Definition + Example + Use Case
