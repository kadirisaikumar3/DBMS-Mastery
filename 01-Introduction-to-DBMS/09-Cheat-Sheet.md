# 📋 Cheat Sheet

DBMS → Database Management System

Database → Collection of Related Data

Table → Rows + Columns

Record → Row

Field → Column

Primary Key → Unique Identifier

Metadata → Data About Data

Examples:

MySQL
Oracle
PostgreSQL
SQL Server

Advantages:

✔ Security

✔ Integrity

✔ Concurrency
