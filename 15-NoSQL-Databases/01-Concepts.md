# 📖 NoSQL Database Concepts

## What is NoSQL?

NoSQL stands for "Not Only SQL."

It refers to databases that do not use the traditional relational table structure.

Instead, they store data using flexible models such as:

- Key-Value
- Document
- Column-Family
- Graph

---

## Why NoSQL?

NoSQL databases are designed for:

- Massive Data Storage
- High Availability
- Horizontal Scaling
- Flexible Schemas
- Fast Performance

---

## Features

- Schema-less
- Distributed
- Highly Scalable
- Fault Tolerant
