# 📘 Chapter 15 - NoSQL Databases

## 📚 Overview

NoSQL (Not Only SQL) databases are designed to handle large volumes of structured, semi-structured, and unstructured data.

Unlike relational databases, NoSQL databases provide high scalability, flexibility, and availability, making them ideal for modern distributed applications.

Examples include MongoDB, Cassandra, Redis, DynamoDB, and Neo4j.
