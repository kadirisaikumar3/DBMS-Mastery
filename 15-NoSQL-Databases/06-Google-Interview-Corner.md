# 🎯 Google Interview Corner

## Most Asked Concepts

### SQL vs NoSQL

SQL

- Fixed Schema
- Tables
- ACID
- Vertical Scaling

NoSQL

- Flexible Schema
- Documents/Key-Value/Graph
- BASE
- Horizontal Scaling

---

### Types of NoSQL Databases

1. Key-Value
   Example: Redis

2. Document
   Example: MongoDB

3. Column-Family
   Example: Cassandra

4. Graph
   Example: Neo4j

---

### CAP Theorem

Consistency

Availability

Partition Tolerance

A distributed system can guarantee only two of these three simultaneously.

---

### BASE

Basically Available

Soft State

Eventually Consistent

---

## Interview Tip

Remember:

SQL → ACID
