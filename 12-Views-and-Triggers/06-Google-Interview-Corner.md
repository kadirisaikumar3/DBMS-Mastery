# 🎯 Google Interview Corner

## Most Asked Concepts

### View

Virtual Table

Created using SQL Query.

---

### Why Use Views?

- Security
- Simplicity
- Abstraction

---

### Trigger

Automatic Action

Triggered by:

INSERT

UPDATE

DELETE

---

### BEFORE Trigger

Runs before operation.

---

### AFTER Trigger

Runs after operation.

---

### Common Interview Question

Difference:

View vs Table

Trigger vs Stored Procedure

---

## Interview Tip

Remember:

Views improve Security.

Triggers improve Automation.
