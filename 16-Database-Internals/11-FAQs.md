# 💬 FAQs

### What are Database Internals?

Mechanisms used internally to store and retrieve data.

---

### What is a Storage Engine?

Component responsible for managing database storage.

---

### What is WAL?

Write-Ahead Logging used for crash recovery.

---

### What is MVCC?

Multi-Version Concurrency Control for concurrent transactions.

---

### What is Buffer Pool?

Memory area used to cache frequently accessed data.

---

### Why is Query Optimizer important?

It selects the most efficient execution plan.
