# 📘 Chapter 9 - Transactions and ACID Properties

## 📚 Overview

A Transaction is a sequence of database operations that are treated as a single logical unit of work.

Transactions ensure that database operations are completed correctly even in the presence of failures.

This chapter covers Transactions, Transaction States, ACID Properties, and their importance in database systems.
