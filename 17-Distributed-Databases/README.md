# 📘 Chapter 17 - Distributed Databases

## 📚 Overview

A Distributed Database is a collection of logically related databases distributed across multiple physical locations but managed as a single database system.

Distributed databases improve scalability, fault tolerance, availability, and performance for large-scale applications.

This chapter covers distributed database architecture, data fragmentation, replication, consistency, distributed transactions, and modern distributed database concepts.
