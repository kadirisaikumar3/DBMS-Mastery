# 📖 Distributed Database Concepts

## What is a Distributed Database?

A Distributed Database stores data across multiple computers connected through a network while appearing as a single database to users.

---

## Why Distributed Databases?

Benefits:

- High Availability
- Scalability
- Fault Tolerance
- Geographic Distribution
- Better Performance

---

## Characteristics

- Multiple Sites
- Shared Data
- Transparent Access
- Distributed Processing

---

## Goals

- Reliability
- Availability
- Performance
