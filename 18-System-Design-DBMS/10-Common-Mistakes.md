# ❌ Common Mistakes

## Mistake 1

Using Vertical Scaling forever.

Hardware has limits.

---

## Mistake 2

Ignoring Replication.

Single database failures can cause downtime.

---

## Mistake 3

Using Cache for every query.

Cache only frequently accessed data.

---

## Mistake 4

Confusing Replication and Sharding.

Replication

↓

Copies Data

Sharding

↓

Splits Data

---

## Mistake 5

Ignoring Failover planning.

Always design for server failures.
