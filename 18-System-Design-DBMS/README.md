# 📘 Chapter 18 - System Design DBMS

## 📚 Overview

Database design is one of the most important parts of System Design.

Choosing the right database architecture determines scalability, availability, reliability, and performance.

This chapter explains how databases are used in large-scale systems such as Google, Amazon, Netflix, Uber, and Facebook.
