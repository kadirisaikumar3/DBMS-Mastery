# 📖 ER Model Concepts

## What is ER Model?

ER Model stands for Entity Relationship Model.

It is a conceptual model used to design databases.

It describes:

- Entities
- Attributes
- Relationships

before creating actual database tables.

---

## Why ER Model?

ER Models help:

- Visualize database structure
- Reduce design errors
- Improve database planning
- Understand relationships

---

## Basic Components

1. Entity
2. Attribute
3. Relationship

---

## Example

College Database

Entities:

- Student
- Course
- Faculty

Relationships:

Student Enrolls In Course

Faculty Teaches Course
