# 📘 Chapter 3 - ER Model

## 📚 Overview

The Entity Relationship (ER) Model is a high-level conceptual data model used to design databases.

It helps represent real-world objects, their attributes, and relationships between them.

ER Models are widely used during database design before creating actual database tables.
