# ❌ Common Mistakes

## Mistake 1

Confusing Entity and Attribute.

Student = Entity

Name = Attribute

---

## Mistake 2

Ignoring Cardinality.

Cardinality defines relationship limits.

---

## Mistake 3

Missing Relationships.

Entities alone are not enough.

---

## Mistake 4

Using Attributes as Entities.

Wrong:

StudentName = Entity

Correct:

Student = Entity

Name = Attribute
