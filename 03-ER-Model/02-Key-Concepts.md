# 🔑 Key Concepts

## Entity

A real-world object.

Examples:

- Student
- Employee
- Customer

---

## Entity Set

Collection of similar entities.

Example:

All Students

---

## Attribute

Property of an entity.

Example:

Student

- ID
- Name
- Age

---

## Relationship

Association between entities.

Example:

Student Enrolls In Course

---

## Cardinality

Defines how many entities can participate.

Examples:

- One-to-One
- One-to-Many
- Many-to-One
- Many-to-Many

---

## ER Diagram

Graphical representation of database design.
