# 🎯 Google Interview Corner

## Most Asked Concepts

### 1-Tier Architecture

Used mostly for learning and local systems.

---

### 2-Tier Architecture

Client communicates directly with Database Server.

Examples:

- Desktop Applications
- Small Business Applications

---

### 3-Tier Architecture

Most commonly used architecture.

Structure:

Presentation Layer
      ↓
Application Layer
      ↓
Database Layer

Examples:

- Google
- Amazon
- Netflix
- Facebook

---

## Interview Tip

Whenever asked:

"Which architecture do modern applications use?"

Answer:

3-Tier Architecture

Reason:

- Security
- Scalability
- Maintainability
