## 🏗️ Types of Database Architecture

### 1-Tier Architecture

User, DBMS, and Database reside on the same system.

Example:

- MS Access
- Local Database Systems

---

### 2-Tier Architecture

Client communicates directly with Database Server.

Example:

- Desktop Applications
- Small Business Applications

---

### 3-Tier Architecture

Presentation Layer
↓
Application Layer
↓
Database Layer

Examples:

- Google
- Amazon
- Netflix
- Facebook
