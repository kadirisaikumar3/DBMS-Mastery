# 🔑 Key Concepts

## Tier

A layer in system architecture.

---

## Client

User application requesting data.

Examples:

- Browser
- Mobile App
- Desktop Application

---

## Server

Provides requested services.

Examples:

- MySQL Server
- Oracle Server

---

## Database

Stores data permanently.

---

## DBMS

Manages communication between users and database.

---

## Data Independence

Ability to modify schema without affecting applications.

---

## Scalability

Ability to handle increasing users and data.
