# 📖 Database Architecture

## What is Database Architecture?

Database Architecture defines how data is stored, managed, and accessed in a DBMS.

It describes the interaction between:

- Users
- Applications
- DBMS
- Database

---

## Why Architecture is Important?

- Data Security
- Scalability
- Performance
- Easy Maintenance
- Data Independence

---

## Types of DBMS Architecture

1. 1-Tier Architecture
2. 2-Tier Architecture
3. 3-Tier Architecture

---

## Goal

To provide efficient communication between users and databases while maintaining security and performance.
