# 🔑 Key Concepts

## Authentication

Verifies user identity.

Example:

Username + Password

---

## Authorization

Determines permissions.

Example:

Read Only Access

---

## Access Control

Restricts database operations.

---

## Encryption

Converts data into unreadable form.

---

## SQL Injection

Attack that injects malicious SQL code.

Example:

' OR '1'='1

---

## Auditing

Tracks database activities.

---

## Backup

Creates copies of data for recovery.
