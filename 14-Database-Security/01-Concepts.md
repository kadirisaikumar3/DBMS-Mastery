# 📖 Database Security Concepts

## What is Database Security?

Database Security is the process of protecting databases from unauthorized access and attacks.

---

## Why Database Security?

Databases store:

- Customer Data
- Financial Data
- Medical Records
- Personal Information

---

## Security Goals

1. Confidentiality
2. Integrity
3. Availability
