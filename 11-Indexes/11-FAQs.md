# 💬 FAQs

### What is an Index?

Data structure used to speed up data retrieval.

---

### Why are Indexes important?

Improve query performance.

---

### What is a Clustered Index?

Determines physical order of data.

---

### What is a Non-Clustered Index?

Separate structure containing pointers.

---

### What is a Composite Index?

Index built on multiple columns.

---

### Which Index structure is most common?

B-Tree.
