# ❌ Common Mistakes

## Mistake 1

Confusing Candidate Key and Primary Key.

Candidate Keys can be many.

Primary Key is only one.

---

## Mistake 2

Thinking Foreign Key must be unique.

Foreign Keys can contain duplicate values.

---

## Mistake 3

Using NULL in Primary Key.

Primary Keys cannot contain NULL values.

---

## Mistake 4

Ignoring Constraints.

Constraints maintain data integrity.
