# 📘 Chapter 5 - Keys and Constraints

## 📚 Overview

Keys and Constraints are fundamental concepts in relational databases.

Keys help uniquely identify records, while constraints enforce data integrity and consistency.

This chapter covers all important keys and constraints used in database design.
