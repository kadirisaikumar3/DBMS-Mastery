# 🔑 Key Concepts

## 1NF

Rules:

- Atomic values only
- No repeating groups

---

## 2NF

Rules:

- Must be in 1NF
- No Partial Dependency

---

## 3NF

Rules:

- Must be in 2NF
- No Transitive Dependency

---

## BCNF

Rules:

- Stronger version of 3NF
- Every determinant must be a Candidate Key

---

## Denormalization

Combining tables to improve query performance.
