# 📝 Solved Examples

## Example 1

Question:

Retrieve all students.

Answer:

SELECT *
FROM Students;

---

## Example 2

Question:

Find students older than 20.

Answer:

SELECT *
FROM Students
WHERE Age > 20;

---

## Example 3

Question:

Sort students by age.

Answer:

SELECT *
FROM Students
ORDER BY Age;

---

## Example 4

Question:

Show only unique cities.

Answer:

SELECT DISTINCT City
FROM Students;
