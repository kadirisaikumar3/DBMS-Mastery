# 💬 FAQs

### What is SQL?

Language used to interact with relational databases.

---

### What is SELECT?

Used to retrieve data.

---

### What is WHERE?

Used to filter rows.

---

### What is ORDER BY?

Used to sort rows.

---

### What is GROUP BY?

Used to group similar records.

---

### What is HAVING?

Used to filter grouped data.
