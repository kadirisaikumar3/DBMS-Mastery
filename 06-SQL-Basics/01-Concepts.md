# 📖 SQL Basics

## What is SQL?

SQL stands for Structured Query Language.

It is used to communicate with relational databases.

---

## Why SQL?

SQL helps:

- Create Databases
- Create Tables
- Insert Records
- Update Records
- Delete Records
- Retrieve Records

---

## Example

Student Table

| ID | Name | Age |
|----|------|-----|
| 1 | Sai | 23 |
| 2 | John | 22 |

SQL Query:

SELECT * FROM Student;
