# 📘 Chapter 6 - SQL Basics

## 📚 Overview

SQL (Structured Query Language) is the standard language used to interact with relational databases.

SQL allows users to:

- Store Data
- Retrieve Data
- Update Data
- Delete Data
- Manage Database Objects

Examples:

- MySQL
- PostgreSQL
- Oracle
- SQL Server
