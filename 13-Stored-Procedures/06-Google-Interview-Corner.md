# 🎯 Google Interview Corner

## Most Asked Concepts

### Stored Procedure

Precompiled SQL statements stored inside database.

---

### Why Use Stored Procedures?

- Better Performance
- Reusability
- Security
- Maintainability

---

### Input Parameters

Pass values into procedure.

Example:

StudentID

---

### Output Parameters

Return values from procedure.

Example:

TotalStudents

---

### Common Interview Question

Difference:

Stored Procedure

vs

Function

---

## Interview Tip

Remember:

Stored Procedure

→ Can perform actions

Function

→ Returns a value
