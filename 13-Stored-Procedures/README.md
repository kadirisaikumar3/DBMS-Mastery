# 📘 Chapter 13 - Stored Procedures

## 📚 Overview

Stored Procedures are precompiled SQL statements stored inside the database.

They improve performance, reusability, maintainability, and security.

Stored Procedures allow developers to execute complex business logic directly inside the database.
