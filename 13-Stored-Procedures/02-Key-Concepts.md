# 🔑 Key Concepts

## Stored Procedure

Precompiled SQL code.

---

## Input Parameter

Accepts values from user.

Example:

StudentID

---

## Output Parameter

Returns values.

Example:

TotalStudents

---

## Procedure Execution

EXEC ProcedureName

---

## Procedure Benefits

- Fast Execution
- Reusability
- Security
- Maintainability

---

## Procedure Limitation

Database-specific syntax.
