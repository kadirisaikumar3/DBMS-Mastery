# 💬 FAQs

### What is a Stored Procedure?

A collection of SQL statements stored inside a database.

---

### Why use Stored Procedures?

Performance, Security, Reusability.

---

### How do you execute a Procedure?

EXEC ProcedureName;

---

### Can Procedures accept parameters?

Yes.

Input and Output Parameters.

---

### Can Procedures return values?

Yes.

Using Output Parameters.

---

### Are Stored Procedures faster than normal queries?

Generally yes, because they are precompiled.
