# 🔑 Key Concepts

## Relation

A table in a database.

Example:

Student Table

---

## Tuple

A row in a table.

Example:

(1, Sai, 23)

---

## Attribute

A column in a table.

Examples:

ID

Name

Age

---

## Domain

Set of valid values for an attribute.

Example:

Age → 1 to 120

---

## Schema

Structure of a relation.

Example:

Student(ID, Name, Age)

---

## Degree

Number of attributes in a relation.

Example:

Student(ID, Name, Age)

Degree = 3

---

## Cardinality

Number of tuples in a relation.

Example:

5 rows

Cardinality = 5
