# 📖 Relational Model Concepts

## What is Relational Model?

The Relational Model organizes data into tables.

Each table contains:

- Rows
- Columns

The table is called a Relation.

---

## Example

Student Table

| ID | Name | Age |
|----|------|-----|
| 1 | Sai | 23 |
| 2 | John | 22 |

## Why Relational Model?

- Simplicity
- Flexibility
- Easy Querying
- Data Integrity
