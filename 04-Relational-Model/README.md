# 📘 Chapter 4 - Relational Model

## 📚 Overview

The Relational Model is the most widely used database model.

It organizes data into tables (relations) consisting of rows and columns.

This model was proposed by E.F. Codd in 1970 and forms the foundation of modern relational databases.

Examples:

- MySQL
- PostgreSQL
- Oracle
- SQL Server
